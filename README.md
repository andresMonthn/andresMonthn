## 🏥 Proyecto Principal

### Optisave

Optisave es una plataforma diseñada para optimizar la operación de consultorios y clínicas mediante una solución integral que incluye:

- Agenda médica
- Expediente clínico electrónico
- Gestión de pacientes
- CRM
- Inventario
- Reportes
- Automatización mediante WhatsApp

Mi objetivo es construir una plataforma sólida, moderna y preparada para evolucionar durante los próximos años.
